Step 1: Install a package locked at version
> npm i react@15.6.0 -E --save

Step 2: Demo upgrade does nothing
> npm upgrade

Step 3: Install same package unlocked (notice package-lock.json didn't change)
> npm i react@15.6.0 --save

Step 4: Demo upgrade does something
> npm upgrade
=> React now at 15.6.2

Step 5: Install react 15.5.0 to demo ~ and ^ locking
> npm i react@15.5.0 -E --save

Step 6: Demo ~ bound package upgrade
Change "15.5.0" to "~15.5.0"
> npm upgrade
=> React 15.5.4 installed, i.e. patch version locked

Step 7: Demo ^ bound package upgrade
Changed "~15.5.4" to "^15.5.4"
> npm upgrade
=> React 15.6.2 installed, i.e. minor + patch version locked
